# Real-Time Vehicle Tracking with Circular Geofence

## Features
- Real-time GPS tracking (mobile browser)
- Secure HTTPS using ngrok
- FastAPI backend
- Haversine distance calculation
- Circular geofence
- Entry / Exit event detection
- Fleet dashboard with Leaflet map

## Architecture
Phone → HTTPS → FastAPI → Geofence Logic → Dashboard

## Tech Stack
- FastAPI
- Streamlit
- Folium (Leaflet)
- ngrok
